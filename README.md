# Studpay

## Краткая характеристика проекта

«StudPay» — это веб-ориентированная информационная система, предназначенная для
организации безопасного и удобного процесса купли-продажи, а также обмена товарами и
услугами между студентами. Система автоматизирует процессы размещения объявлений,
поиска, взаимодействия между пользователями (продавцами и покупателями), проведения
безопасных сделок и модерации контента. Платформа обеспечивает структурированное
хранение информации о пользователях (с верификацией по принадлежности к вузу),
товарах, заказах и транзакциях.

## Описание проекта

Проект разработан в рамках учебного задания согласно техническому заданию заказчика.

Система представляет собой веб-приложение, состоящее из серверной и клиентской части.  
Backend реализован на ASP.NET Core с использованием PostgreSQL.  
Frontend реализован на React (Vite) + .

Взаимодействие между клиентом и сервером осуществляется через REST API.  
Авторизация реализована на основе cookie.

Проект разворачивается с использованием Docker и Docker Compose.

## Запуск проекта

1. Создать файл `.env` в корне проекта.
2. Указать необходимые переменные окружения:
   - параметры подключения к базе данных
   - ASPNETCORE_ENVIRONMENT
   - VITE_API_URL
3. Выполнить команду:

```bash
docker-compose up --build -d
```
